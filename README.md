# Site_IP
Proiect pentru IP

# Utilizare:
- Pornire server `npm run devStart`
- Adresa paginii `192.168.83.130:3000`

# Cerinte sistem
- Windows 7 32bit Service Pack 1

# Programe necesare
- python
- npm
- mysql database

# Note
- baza de date folosita se numeste mjpm si nu are parola
- conexiunea se realizeaza la `localhost:3306`
